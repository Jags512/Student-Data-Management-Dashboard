# Student Data Analysis & Dashboard

## Overview
This project analyzes student data to evaluate academic performance and attendance trends.

## Tools Used
- Python (Pandas)
- SQL
- Streamlit Dashboard

## Features
- Data Cleaning
- KPI Analysis
- Interactive Dashboard

## Run Project

1. Install requirements:
pip install -r requirements.txt

2. Run dashboard:
streamlit run dashboard/dashboard.py
